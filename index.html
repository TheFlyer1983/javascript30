JavaScript 30

JavaScript30 is a free Javascript course by Wes Bos (Javascript30). This course focuses on using vanilla javascript, that means no frameworks. No React, No Angular, No Vue, just plain JavaScript, albeit using some of the more modern ES6 functionality.

This page is a showcase of all of the lessons that make up the JavaScript30 course. 30 short daily activities (the longest video is around 30 minutes), to embed what you have previously learned about Javascript.

All my source files are available on my github repository here: GitHub Repo, but the course is free so rather than looking at my code, I would recommend taking the course.
